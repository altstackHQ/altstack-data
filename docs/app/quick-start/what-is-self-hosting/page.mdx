# What is Self-Hosting?

**In one sentence:** Self-hosting means running software on a server *you* control, instead of paying someone else to run it for you.

That's it. That's the whole concept.

When you use Gmail, Google runs the email server. When you use Notion, Notion Inc runs the database. When you self-host, **you** run these things â€” on a cheap VPS, a spare laptop, or even a Raspberry Pi.
